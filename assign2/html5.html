HTML 5 New Features

How to focus on an entry

autofocus 
<input type="text" name="entry1" id="entry1" autofocus="autofocus" /> autofocus

How to set a placeholder

placeholder 
<input type="text" name="entry2" id="entry2" placeholder="Enter Something" / > placeholder

How to set required for an entry

required 
<input type="text" name="entry3" id="entry3" required="required" /> required

How to create a list/datalist

list/datalist 
<input type="text" name="entry4" id="entry4" list="entry4list" /> list/datalist
<datalist id="entry4list">
	<option value="Android">
	<option value="iOS">
	<option value="Symbian">
	<option value="Windows Phone">
	<option value="Ubuntu">
</datalist>

How to ensure a valid email address format

Email 
<input type="email" name="entry5" id="entry5" /> Email

How to ensure a url format

URL 
<input type="url" name="entry6" id="entry6" /> URL

HTML5 New attribute/value

  • autofocus="autofocus"
  • placeholder="Enter Something"
  • required="required"
  • list="entry4list"
  • type="email"
  • type="url"
