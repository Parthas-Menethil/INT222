HTML5

Today, although HTML5 has not been fully supported by most browsers, it is the potential new trend in the web world. HTML5 reinforces the performance of web pages and supports local database communication. With CSS and JavaScript, HTML5 can make a web page function as a powerful web software. By now, it is still developing and has a long way to go.

Reset CSS

Almost every browser has its own built-in CSS for interpreting HTML pages. These built-in CSS might be different among browsers. Reset css can disable the broswer's built-in CSS, so programmers can control the browser how to display the HTML page. Sometimes, even though the reset CSS is used, HTML page may also look diverse in different browsers.
